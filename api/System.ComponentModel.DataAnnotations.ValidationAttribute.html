Gets the localized validation error message.

Remarks

The error message string is obtained by evaluating the ErrorMessage property or by evaluating the ErrorMessageResourceType and ErrorMessageResourceName properties. The two cases are mutually exclusive. The second case is used if you want to display a localized error message.

Validate(Object, ValidationContext)

Validates the specified object.

Remarks

This method calls the IsValid(Object) method to determine whether the object that is specified by the value parameter is valid. If the method returns false, this method calls the FormatErrorMessage(String) method to get a localized message that states the problem. It then throws a ValidationException exception.

Credentials

The credentials to submit to the proxy server for authentication.

Remarks

The Credentials property is an ICredentials instance that contains the authorization credentials to send to the proxy server in response to an HTTP 407 (proxy authorization) status code.

IsBypassed(Uri)

Indicates that the proxy should not be used for the specified host.

Remarks

The IsBypassed(Uri) method indicates whether to use the proxy server to access the host that is specified in the host parameter. If IsBypassed(Uri) is true, the proxy is not used to contact the host and the request is passed directly to the server.

Dispose()

Releases the resources used by the current instance of the EventListener class.

Remarks

Call Dispose when you are finished using the EventListener. The Dispose method leaves the EventListener in an unusable state. After calling Dispose, you must release all references to the EventListener so the garbage collector can reclaim the memory that the EventListener was occupying.

For more information, see Cleaning Up Unmanaged Resources and Implementing a Dispose Method.

EventSourceIndex(EventSource)

Gets a small non-negative number that represents the specified event source.

Remarks

The value returned by EventSourceIndex(EventSource) identifies an event source that is suitable for indexing an array. This integer is unique within the application domain. An event listener might find it useful to store additional information about each event source that is connected to it. The value returned by EventSourceIndex(EventSource) allows this extra information to be efficiently stored in a expandable array such as List.

OnEventSourceCreated(EventSource)

Called for all existing event sources when the event listener is created and when a new event source is attached to the listener.

Remarks

Event source implementations should be aware that these callbacks are issued for existing event sources when the event listener is instantiated.

In a multithreaded environment, it is possible for OnEventWritten(EventWrittenEventArgs) callbacks to occur before the OnEventSourceCreated(EventSource) callback.

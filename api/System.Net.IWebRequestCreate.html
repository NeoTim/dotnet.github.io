Interface IWebRequestCreate

Provides the base interface for creating WebRequest instances.

Namespace:System.Net
Assembly:System.Net.Requests.dll
Syntax
public interface IWebRequestCreate
Remarks

The IWebRequestCreate interface defines the method that WebRequest descendants must use to register with the Create(String) method.

Classes that implement the IWebRequestCreate interface can be registered with the WebRequest class and associated with a specific URI scheme so that the WebRequest calls the class's Create(Uri) method when a URI that matches that scheme is requested.

Create(Uri)

Creates a WebRequest instance.

Remarks

The Create(Uri) method must return an initialized instance of the WebRequest descendant that is capable of performing a standard request/response transaction for the protocol without needing any protocol-specific fields modified.

Parameters
Type Name Description
Uri sourceUri

The uniform resource identifier (URI) of the PackagePart or the Package (SourceUri="/") that owns the relationship.
PackageRelationshipSelectorType selectorType

The type of the selectionCriteria, either by relationship Id or relationship Type.
String selectionCriteria

The qualification string that is used to select the relationships based on the selectorType.
Exceptions
Type Condition
ArgumentNullException

sourceUri or selectionCriteria is null.
ArgumentOutOfRangeException

The selectorType parameter is not valid.
XmlException

The selectorType parameter is Id but selectionCriteria is not a valid XML Schema Definition (XSD) identifier (ID).
ArgumentException

The selectionCriteria is not valid for the specified selectorType.
ArgumentException

The sourceUri is not the Package root ("/") and is also not a valid PackagePart URI.

Order

Gets or sets the zero-based order of the column the property is mapped to.

Remarks

The specified order is relative to the order values specified on other properties in the class. For example, given the following definition, the column Line1 appears before Line2, and Line2 appears before Country.

public class Address
{
[Column(Order = 1)]
public string Line1 { get; set; }
[Column(Order = 4)]
public string Line2 { get; set; }
[Column(Order = 17)]
public string Country { get; set; }
}

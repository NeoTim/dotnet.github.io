Gets the number of key/value collection pairs in the Lookup.

Remarks

The value of the Count property does not change because items cannot be added to or removed from a Lookup object after it has been created.

Item[TKey]

Gets the collection of values indexed by the specified key.

Remarks

This indexed property provides the ability to index a specific collection of values in the Lookup by using the following syntax: myLookup[key] in Visual C# or myLookup(key) in Visual Basic. If the key is not found in the collection, an empty sequence is returned.

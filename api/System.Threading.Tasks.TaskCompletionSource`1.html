Class TaskCompletionSource<TResult>

Represents the producer side of a Task unbound to a delegate, providing access to the consumer side through the Task property.

Task

Gets the Task created by this TaskCompletionSource.

Remarks

This property enables a consumer to access the Task that is controlled by this instance. When you create a TaskCompletionSource object, the Status property of this Task object returns WaitingForActivation

The SetResult(TResult), SetException(Exception), SetException(IEnumerable), and SetCanceled() methods (and their "Try" variants) on this instance all result in the relevant state transitions on this underlying Task.

TrySetCanceled(CancellationToken)

Attempts to transition the underlying Task into the Canceled state and enables a cancellation token to be stored in the canceled task.

Remarks

The method returns false if the underlying Task object is already in one of the following three final states.

  • Canceled

  • Faulted

  • RanToCompletion

This method also returns false if the underlying Task object has already been disposed.

Class AssemblyFlagsAttribute

Specifies a bitwise combination of AssemblyNameFlags flags for an assembly, describing just-in-time (JIT) compiler options, whether the assembly is retargetable, and whether it has a full or tokenized public key. This class cannot be inherited.

Syntax
[AttributeUsage(AttributeTargets.Assembly, Inherited = false)]
public sealed class AssemblyFlagsAttribute : Attribute
Remarks

The AssemblyNameFlags enumeration describes the assembly characteristics that can be set using this attribute.

To access the flags that have been specified for an assembly, use the property to obtain an AssemblyName object, then use the Flags property to obtain an AssemblyNameFlags value.

To specify AssemblyNameFlags flags for a dynamic assembly, set the Flags property of the AssemblyName object that you pass to the method.

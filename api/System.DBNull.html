Value

Represents the sole instance of the DBNull class.

Remarks

DBNull is a singleton class, which means only this instance of this class can exist.

If a database field has missing data, you can use the Value property to explicitly assign a DBNull object value to the field. However, most data providers do this automatically.

To evaluate database fields to determine whether their values are DBNull, you can pass the field value to the DBNull.Value.Equals method. However, this method is rarely used because there are a number of other ways to evaluate a database field for missing data. These include the Visual Basic IsDBNull function, the System.Convert.IsDBNull(System.Object) method, the System.Data.DataTableReader.IsDBNull(System.Int32) method, the IsDBNull(Int32) method, and several other methods.

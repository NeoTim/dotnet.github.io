Examples

The following code example calculates binomial coefficients, which are values used in probability and statistics. This example uses a DefaultTraceListener to trace results and log errors. It creates a new DefaultTraceListener, adds it to the Listeners collection, and sets the System.Diagnostics.DefaultTraceListener.LogFileName property to the log file specified in the command-line arguments.

If an error is detected while processing the input parameter, or if the CalcBinomial function throws an exception, the method logs and displays an error message. If the System.Diagnostics.DefaultTraceListener.AssertUiEnabled property is false, the error message is also written to the console. When the result is calculated successfully, the Write(String) and WriteLine(String) methods write the results to the log file.

The , , and methods cause trace information to be written only to the DefaultTraceListener. To write trace information to all listeners in the Listeners collection, use the , , and methods of the Trace class.

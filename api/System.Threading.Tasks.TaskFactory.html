ContinueWhenAll(Task[], Action<Task[]>)

Creates a continuation task that starts when a set of specified tasks has completed.

Remarks

The ContinueWhenAll(Task[], Action) method executes the continuationAction delegate when all tasks in the tasks array have completed, regardless of their completion status.

Exceptions thrown by tasks in the tasks array are not available to the continuation task through structured exception handling. You can determine which exceptions were thrown by examining the Exception property of each task in the tasks array. To use structured exception handling to handle exceptions thrown by tasks in the tasks array, call the WaitAll(Task[]) method.

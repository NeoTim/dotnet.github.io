Class AbandonedMutexException

The exception that is thrown when one thread acquires a Mutex object that another thread has abandoned by exiting without releasing it.

Mutex

Gets the abandoned mutex that caused the exception, if known.

Remarks

If the exception is thrown on a call to the method, the Mutex returned is the abandoned mutex with the lowest index in the array of WaitHandle objects that was passed to , or null if the abandoned mutex could not be identified. If the exception is thrown on a call to or , this property always returns null.

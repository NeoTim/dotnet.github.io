MethodName

Gets the name of a method that will return a list of types that should be recognized during serialization or deserialization.

Remarks

The method must exist on the class or structure that the KnownTypeAttribute is applied to, must be static, must accept no parameters, and must return an instance of any type that implements the generic IEnumerable interface, such as the Collection class, or an array of Type objects.

The method is called once per application domain when the data contract is loaded for the type.

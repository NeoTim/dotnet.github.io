SendAsync(HttpRequestMessage, CancellationToken)

Send an HTTP request as an asynchronous operation.

Remarks

This operation will not block. The returned Task object will complete once the entire response including content is read.

Most applications that are connecting to a web site will use one of the SendAsync(HttpRequestMessage) methods on the HttpClient class.

ValidationContext(Object, IServiceProvider, IDictionary<Object, Object>)

Initializes a new instance of the ValidationContext class using the service provider and dictionary of service consumers.

Remarks

The serviceProvider parameter represents a service that can be used by the GetService(Type) method in order to perform custom validation.

If the items parameter is null, an empty dictionary is created. If the parameter is not null, the set of key/value pairs is copied into a new dictionary, which prevents the service consumers from modifying the original dictionary.

GetService(Type)

Returns the service that provides custom validation.

Remarks

If the System.ComponentModel.DataAnnotations.ValidationContext.ServiceContainer object is defined at the application level, the GetService(Type) method accesses it to retrieve the requested service.

If the System.ComponentModel.DataAnnotations.ValidationContext.ServiceContainer object is not defined or it does not return the service, the GetService(Type) method queries the IServiceProvider object that is passed to the System.ComponentModel.DataAnnotations.ValidationContext. method in order to obtain the service.

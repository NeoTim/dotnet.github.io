ChainElements

Gets a collection of X509ChainElement objects.

Remarks

Each X509ChainElement object is a representation of an element in the chain that was built during a call to the Build(X509Certificate2) method.

A chain element consists of an X509Certificate2 object, an X509ChainStatus structure, and an extra information string (available only on Windows XP platforms).

Build(X509Certificate2)

Builds an X.509 chain using the policy specified in X509ChainPolicy.

Remarks

Note that a valid X.509 certificate from a trusted issuer is valid only for the use specified in the X509ChainPolicy object. Certificates that meet these chain policy rules might still be invalid for specific uses with such features as Security/MIME (SMIME), Authenticode, or Secure Sockets Layer (SSL). If further processing is required to determine whether the certificate is valid for a specific policy, derive a class from X509Chain and override the Build(X509Certificate2) method so that it calls the base class Build method first, and then does the additional processing.

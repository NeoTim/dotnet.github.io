Dispose()

Releases all resources used by the current instance of the CancellationTokenRegistration class.

Remarks

Call Dispose when you are finished using the CancellationTokenRegistration. The Dispose method leaves the CancellationTokenRegistration in an unusable state. After calling Dispose, you must release all references to the CancellationTokenRegistration so the garbage collector can reclaim the memory that the CancellationTokenRegistration was occupying.

For more information, see Cleaning Up Unmanaged Resources and Implementing a Dispose Method.

Always call Dispose before you release your last reference to the CancellationTokenRegistration. Otherwise, the resources it is using will not be freed until the garbage collector calls the CancellationTokenRegistration object's Finalize method.

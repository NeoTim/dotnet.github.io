Enum MemberTypes

Marks each type of member that is defined as a derived class of MemberInfo.

Namespace:System.Reflection
Assembly:System.Reflection.dll
Syntax
public enum MemberTypes : int
Remarks

These enumeration values are returned by the following properties:

  • System.Type.MemberType.

  • System.Reflection.MemberInfo.MemberType.

  • ConstructorInfo.

  • EventInfo.

  • FieldInfo.

  • MethodInfo.

  • PropertyInfo.

To obtain the MemberTypes value for a type:

  1. Get a Type object that represents that type.

  2. Retrieve the value of the System.Type.MemberType property.

To get the MemberTypes values for the members of a type.:

  1. Get a Type object that represents that type.

  2. Retrieve the MemberInfo array that represents the members of that type by calling the System.Type.GetMembers method.

  3. Retrieve the value of the From the System.Reflection.MemberInfo.MemberType property for each member in the array. A switch statement in C# or Select Case statement in Visual Basic is typically used to process member types.

MemberTypes matches CorTypeAttr as defined in the corhdr.h file.

ActivityId

[Supported in the .NET Framework 4.5.1 and later versions]

Gets the activity ID on the thread that the event was written to.

Remarks

Retrieving the value of the ActivityId property accesses thread local storage. This has the following implications:

  • The ActivityId field is valid only for the duration of the OnEventWritten(EventWrittenEventArgs) callback.

  • If the activity ID is changed during the callback, the activity ID returned by the ActivityId property is invalid.

Class HttpMethod

A helper class for retrieving and comparing standard HTTP methods and for creating new HTTP methods.

Inheritance
Object
HttpMethod
Namespace:System.Net.Http
Assembly:System.Net.Http.dll
Syntax
public class HttpMethod : IEquatable<HttpMethod>
Remarks

The most common usage of HttpMethod is to use one of the static properties on this class. However, if an app needs a different value for the HTTP method, the System.Net.Http.HttpMethod. constructor initializes a new instance of the HttpMethod with an HTTP method that the app specifies.

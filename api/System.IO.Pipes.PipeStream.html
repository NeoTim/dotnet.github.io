Read(Byte[], Int32, Int32)

Reads a block of bytes from a stream and writes the data to a specified buffer.

Remarks

Use the CanRead property to determine whether the current PipeStream object supports read operations.

Calling the Read(Byte[], Int32, Int32) method blocks until count bytes are read or the end of the stream is reached. For asynchronous read operations, see System.IO.Pipes.PipeStream.BeginRead(System.Byte[],System.Int32,System.Int32,System.AsyncCallback,System.Object) and System.IO.Pipes.PipeStream.EndRead(System.IAsyncResult).

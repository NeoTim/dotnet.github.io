Interface IGrouping<TKey, TElement>

Represents a collection of objects that have a common key.

Examples

The following example demonstrates how to work with an IGrouping object.

In this example, GroupBy(IEnumerable, Func) is called on the array of MemberInfo objects returned by GetMembers(). GroupBy(IEnumerable, Func) groups the objects based on the value of their System.Reflection.MemberInfo.MemberType property. Each unique value for System.Reflection.MemberInfo.MemberType in the array of MemberInfo objects becomes a key for a new IGrouping object, and the MemberInfo objects that have that key form the IGrouping object's sequence of values.

Finally, the First(IEnumerable) method is called on the sequence of IGrouping objects to obtain just the first IGrouping object.

The example then outputs the key of the IGrouping object and the Name property of each value in the IGrouping object's sequence of values. Notice that to access an IGrouping object's sequence of values, you simply use the IGrouping variable itself.

Examples

The following example demonstrates how to use the Key property to label each IGrouping object in a sequence of IGrouping objects. The GroupBy(IEnumerable, Func) method is used to obtain a sequence of IGrouping objects. The foreach in Visual C# or For Each in Visual Basic loop then iterates through each IGrouping object, outputting its key and the number of values it contains.

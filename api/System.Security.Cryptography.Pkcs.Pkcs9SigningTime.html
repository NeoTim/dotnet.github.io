Class Pkcs9SigningTime

The Pkcs9SigningTime class defines the signing date and time of a signature. A Pkcs9SigningTime object can be used as an authenticated attribute of a System.Security.Cryptography.Pkcs.CmsSigner object when an authenticated date and time are to accompany a digital signature.

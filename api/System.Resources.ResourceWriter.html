Class ResourceWriter

Generate()

Saves all resources to the output stream in the system default format.

Remarks

The Generate() method is called implicitly by the System.Resources.ResourceWriter.Close method if it is not called by your application code.

Generate() can only be called once, after all calls to AddResource(String, String) and System.Resources.ResourceWriter.AddResourceData(System.String,System.String,System.Byte[]) have been made. If an exception occurs while writing the resources, the output stream will be closed to prevent writing invalid information.

Generate() does not close the output stream in normal cases. Unless you are combining extra data with your .resources file or need access to the stream afterwards, you should call System.Resources.ResourceWriter.Close after calling Generate(), or simply call System.Resources.ResourceWriter.Close.

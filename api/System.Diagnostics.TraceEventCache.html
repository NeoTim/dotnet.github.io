Class TraceEventCache

Provides trace event data specific to a thread and a process.

Inheritance
Object
TraceEventCache
Namespace:System.Diagnostics
Assembly:System.Diagnostics.TraceSource.dll
Syntax
public class TraceEventCache : object
Remarks

The TraceEventCache class, designed as a performance optimization for trace listener calls, is of interest only to developers creating custom trace listeners.

The TraceEventCache class is used as a parameter in tracing methods to accurately identify the source of a trace event. Examples of methods that use TraceEventCache are TraceEvent(TraceEventCache, String, TraceEventType, Int32, String) and ShouldTrace(TraceEventCache, String, TraceEventType, Int32, String, Object[], Object, Object[]). The System.Diagnostics.TraceEventCache.LogicalOperationStack property contains data that can be used to correlate the trace with related traces.

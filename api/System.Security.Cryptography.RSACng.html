Key

Gets the key that will be used by the RSACng object for any cryptographic operation that it performs.

Remarks

The returned CngKey object is disposed if the key is reset, for instance by changing the KeySize property, by using the ImportParameters(RSAParameters) to create a new key, or by disposing of the parent RSA object. Therefore, you should ensure that the key object is no longer used in these cases.

This CngKey object is not the same as the CngKey object passed to the System.Security.Cryptography.RSACng. constructor, if that constructor was used. However, it will point to the same CNG key.

Replaces the existing key that the current instance is working with by creating a new CngKey for the parameters structure.

If parameters contains only an exponent and modulus, only a public key is imported. If parameters also contains P and Q values, a full key pair is imported.

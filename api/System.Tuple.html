Create<T1, T2, T3, T4, T5, T6, T7, T8>(T1, T2, T3, T4, T5, T6, T7, T8)

Creates a new 8-tuple, or octuple.

Remarks

Create(T1, T2, T3, T4, T5, T6, T7, T8) is a helper method that you can call to instantiate an 8-tuple without having to explicitly specify the types of its components.

You must call the System.Tuple`8. constructor to create a tuple with nine or more components unless your language provides a special syntax for this purpose. The static (Shared in Visual Basic) methods of the Tuple class cannot be used to create a tuple with nine or more components.

Remarks

Create(T1, T2, T3, T4, T5, T6, T7, T8) is a helper method that you can call to instantiate an 8-tuple without having to explicitly specify the types of its components.

You must call the System.Tuple`8. constructor to create a tuple with nine or more components unless your language provides a special syntax for this purpose. The static (Shared in Visual Basic) methods of the Tuple class cannot be used to create a tuple with nine or more components.

Examples

The following example creates an 8-tuple whose components are prime numbers that are less than 20.

System.Tuple.Create#17

This is equivalent to the following example, which uses the Tuple class constructor instead of the Create(T1, T2, T3, T4, T5, T6, T7, T8) factory creation method. Note that instantiating a Tuple object in this way involves considerably more code, because you must declare a nested Tuple object as the Tuple object's eighth component to produce an octuple.

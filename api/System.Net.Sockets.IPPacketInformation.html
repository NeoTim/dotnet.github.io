Gets the origin information of the packet that was received as a result of calling the ReceiveMessageFrom(Byte[], Int32, Int32, ref SocketFlags, ref EndPoint, out IPPacketInformation) method or System.Net.Sockets.Socket.EndReceiveMessageFrom(System.IAsyncResult,System.Net.Sockets.SocketFlags@,System.Net.EndPoint@,System.Net.Sockets.IPPacketInformation@) method.

Declaration
public IPAddress Address { get; }
Returns
Type Description
IPAddress

An IPAddress that indicates the origin information of the packet that was received as a result of calling the ReceiveMessageFrom(Byte[], Int32, Int32, ref SocketFlags, ref EndPoint, out IPPacketInformation) method or System.Net.Sockets.Socket.EndReceiveMessageFrom(System.IAsyncResult,System.Net.Sockets.SocketFlags@,System.Net.EndPoint@,System.Net.Sockets.IPPacketInformation@) method. For packets that were sent from a unicast address, the Address property will return the IPAddress of the sender; for multicast or broadcast packets, the Address property will return the multicast or broadcast IPAddress.
